## Problemas possiveis

Caso tenha problemas ao executar o scipt, rode os comandos abaixo

```bash
docker-compose exec bandersnatch ash
chmod g+rx,o+rx script.sh
```

ou

```bash
sudo chmod 777 script.sh
```
